# Security Policy

## Supported versions

This is a template repository, so only the `main` branch is supported. Base new
work on the latest `main`.

## Reporting a vulnerability

Please report vulnerabilities privately. Do not open a public issue.

- Preferred: open a [private security advisory](https://github.com/isyll/go-grpc-starter/security/advisories/new).
- Alternatively, email <isylla@softvalleylabs.com>.

Include steps to reproduce and the affected commit. You will get an
acknowledgement, and a fix or mitigation will be coordinated before any public
disclosure.
